개요

네이버 검색 OpenAPI를 활용한 영화 검색 웹 사이트 개발

영화에 대한 정확한 정보 검색을 위한 장르, 개봉 날짜 등을 선택하고 검색할 영화를 검색하는 웹 사이트

개발 내용

Front-End

  • React를 기반으로 전체적인 웹 사이트 설계 및 구현
  • CSS를 활용해 검색 결과 간략화 및 가시성 향상
  • 네이버 검색 OpenAPI를 통한 간단한 영화 검색 구현
  • React Router 라이브러리를 통한 효율적인 웹 페이지 설계
  • Code Pen 사이트의 CSS 레퍼런스를 통한 사이트 제목 제작

Back-End

  • npm을 사용하여 전반적으로 사용된 패키지 관리
  • Node.js를 사용하여 서버 구축
  • CORS 라이브러리를 통한 OpenAPI의 CORS 정책에 따른 데이터 응답 처리 진행
