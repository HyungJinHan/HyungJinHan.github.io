개요

증가하는 배달 수수료를 각 서비스 사용자들과 함께 분담하기 위한 배달 플랫폼 기획 및 개발

타 배달 앱에서 제공되지 않는 배달 수수료 분담을 위한 그룹화 서비스 개발

개발 내용

Front-End

  • React를 기반으로 전체적인 웹 사이트 설계 및 구현
  • CSS와 SCSS, styled-component 등을 활용하여 모바일 환경에 맞는 UI/UX 설계
  • OpenAPI와 주소 검색 라이브러리를 활용한 정확한 위치 정보 표시
  • React Router 라이브러리를 통한 효율적인 웹 페이지 구성
  • React Swiper 등의 시각적인 라이브러리를 활용하여 이용자에게 효과적인 UI 설계
  • Figma를 통한 UI 설계 및 적용

Back-End

  • npm을 사용하여 전반적으로 사용된 패키지 관리
  • Node.js를 사용하여 서버 구축
  • Express.js 프레임워크를 사용한 http 통신
  • MySQL을 통한 데이터베이스 설계
  • axios 라이브러리를 통한 데이터베이스와의 비동기식 데이터 연동

Project Management

  • GitHub를 이용한 코드 관리 및 협업
  • Notion을 활용한 팀 일정 관리 및 프로젝트 관리법 작성
  • 구글 드라이브 공유를 통한 프로젝트 산출문서 작성 및 관리
